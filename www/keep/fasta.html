  • Sequences are expected to be represented in the standard IUB nucleic acid codes.

    • The nucleic acid codes supported are: 
    A --> adenosine      K --> G T (keto)          B --> C G T        N --> A G C T (any)
C --> cytidine       M --> A C (amino)         D --> A G T
G --> guanine        R --> A G (purine)        H --> A C T
T --> thymidine      S --> C G (strong)        V --> A C G
                     W --> A T (weak)
                     Y --> C T (pyrimidine)
