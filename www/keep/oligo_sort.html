Sort siRNA candidates

By default, all the oligos are sorted by the position. But you can sort the oligos by pattern, GC percentage and thermodynamic values by clicking on them. It's a multi-layer sort: each sort is based above your previous sort.

Sort Criteria
sort name sort order
Position numerical order
Pattern A->B->F
GC percentage numerical order
Thermodynamic values first value:
the difference between 5' end sense and 5' end antisense
